# 🏢 Dự Án Website Giới Thiệu Chung Cư

Hệ thống website quản lý và giới thiệu dự án chung cư/bất động sản. Tích hợp đầy đủ các chức năng quản trị, tìm kiếm, phân trang và liên hệ bằng công nghệ AJAX.

## 🛠 Hướng dẫn Cài đặt 

1. Tải toàn bộ source code về, giải nén và đổi tên thư mục thành `WebBookingHotel`.
2. Di chuyển thư mục đó vào đường dẫn `C:\xampp\htdocs\`.
3. Bật **XAMPP** (Start cả Apache và MySQL).
4. Mở trình duyệt, truy cập `http://localhost/phpmyadmin/`.
5. Tạo một Database mới tinh có tên là: **`chungcu_db`**.
6. Chọn Database vừa tạo, bấm sang tab **Nhập (Import)** và tải lên file **`database.sql`** có sẵn trong thư mục gốc để nạp dữ liệu.

## 🚀 Hướng dẫn Sử dụng
- **Trang chủ khách hàng:** `http://localhost/WebBookingHotel/index.php`
- **Trang quản trị (Admin):** `http://localhost/WebBookingHotel/admin/login.php`
